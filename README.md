## API For 小亦

API For 小亦 一个基于基础编程语言并运行在Github开源免费不限制提供一言数据服务平台. https://api.isoyi.net/

****

"食"用方法:使用规范的 HTTP 响应代码来表示请求结果的正确或错误信息。使用 HTTP 内置的特性，如 HTTP Authentication 和 HTTP 请求方法让接口易于理解。所有的 API 请求都会以规范友好的 JSON 对象格式返回（包括错误信息）。
